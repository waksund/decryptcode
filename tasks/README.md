# Assessment Tasks

This directory contains the detailed task descriptions for the blockchain engineer assessment.

## Task Overview

1. **[Task 1: Smart Contract Development](./task-1-smart-contract.md)**
   - Build a TokenVault smart contract
   - Focus: Solidity, security, best practices

2. **[Task 2: Backend API Configuration](./task-2-backend-api.md)**
   - Configure the prebuilt Node.js API
   - Connect to your deployed smart contract
   - Focus: Configuration, integration, testing

3. **[Task 3: Frontend Integration](./task-3-react-native-frontend.md)**
   - Configure the prebuilt Next.js web app
   - Connect to backend API and wallet
   - Focus: Configuration, integration, testing

## Task Order

We recommend completing the tasks in this order:
1. **Task 1** - Smart Contract (foundation)
2. **Task 2** - Backend Configuration (uses Task 1's contract)
3. **Task 3** - Frontend Configuration (uses Task 2's API)
